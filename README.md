
Forward ray-tracer to make an image a bit like the cover of
the Pink Floyd album _Dark Side of the Moon_.

# Build

Build the renderer using CMake:

    % mkdir build
    % cd build
    % cmake ..
    % make

It compiles on MacOS and Linux.
